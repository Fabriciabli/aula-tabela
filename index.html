Tabela

Aqui vamos ver as tabelas do HTML, um fato curioso: antigamente, a estrutura de um site era basicamente composto por uma tabela.Hoje em dia, com as tags semânticas, as tabelas tem a seu verdadeiro significado de utilização, que é realmente receber valores. Junto com o código disponibilizado, terá a imagem de uma estrutura de tabela, mostrada na aula.

Iremos usar para a estrutura de nossa tabela a tag "table", e suas propriedade: "tr" que significa: table rom(linha); "th" table header (cabeçalho) e a "td" table data (dados), para cirar nossa tabela.
